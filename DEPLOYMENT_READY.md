## 📋 What You Need to Deploy

### Required API Keys

1. **OpenAI API Key** (for resume parsing)
   - Get from: https://platform.openai.com/api-keys
   - Add to `.env` as `APP_OPENAI_API_KEY`

2. **Cloudflare R2 Credentials** (for resume storage)
   - Get from: Cloudflare Dashboard → R2
   - Add to `.env`:
     - `APP_R2_ENDPOINT`
     - `APP_R2_ACCESS_KEY`
     - `APP_R2_SECRET_KEY`
     - `APP_R2_BUCKET=resumes`

3. **JWT Secret** (auto-generated by start script)
   - Or manually: `openssl rand -base64 32`

### Optional (for OAuth login)

4. **Google OAuth2** (optional)
   - Get from: https://console.cloud.google.com
   - Add client ID and secret to `.env`

5. **GitHub OAuth2** (optional)
   - Get from: https://github.com/settings/developers
   - Add client ID and secret to `.env`

## 🐛 Troubleshooting

### Backend won't start
```bash
docker-compose logs backend-api
```
Common: JWT secret too short or missing

### Scraper not running
```bash
docker-compose logs scraper-worker
```
Common: Database connection failed

### Frontend can't connect
Check `NEXT_PUBLIC_API_URL` in `.env`
```bash
docker-compose restart frontend
```

### Resume upload fails
Verify R2 credentials in `.env`
```bash
docker-compose restart backend-api
```
